‚úÖ COMMON QUESTIONS ANSWERED
================================================================================

Q: Are my passwords really secure?
A: Yes. Bcrypt uses 12 rounds of hashing. Takes 2^12 = 4096 iterations.
   Even with supercomputer, can't crack bcrypt-hashed passwords.

Q: What if someone steals my database file?
A: Passwords are hashed (not reversible). But optional: Can encrypt entire
   database file with SQLCipher. Read SECURITY_SETUP_GUIDE.txt.

Q: Can phone users access admin panel?
A: No. API requires login with credentials. Roles enforced on both sides.
   Scanner users only scan QR codes. Admins manage users.

Q: Is my data on cloud?
A: No. Database is on YOUR laptop. You have full control.
   Phone connects via network API, not cloud.

Q: What if I lose the laptop?
A: Recommend: Regular backups of mascan_attendance.db to USB or cloud.
   Even if stolen, database is just a file. Can restore anytime.

Q: Can I change passwords?
A: Yes! Edit .env and change ADMIN_PASSWORD. Restart app.
   New password takes effect immediately.

Q: Can I use with multiple phones?
A: Yes! API server can handle multiple concurrent connections.
   Each phone user logs in with their credentials.

Q: Do I need to buy anything?
A: No. Everything is free and open source.
   Bcrypt, Flask, Python - all free.

Q: Is this a security risk?
A: No. Following industry best practices.
   Same approach used by major companies.
