FOR WEB/PHONE DEPLOYMENT (LATER)
================================================================================

1. Web Browser Access (Easiest):
   - Run API server: python final-project/src/api_server.py
   - Run app: python final-project/src/main.py
   - Open browser: http://localhost:8080
   - Or from other device: http://your-laptop-ip:8080
   - Uses camera display + manual QR entry (no auto-scanning)

2. Build APK for Phone:
   pip install buildozer cython
   buildozer android debug
   Then install on phone: adb install bin\app.apk

3. Configure on Phone:
   - Set API_BASE_URL in remote_config.py to your laptop IP
   - Example: http://192.168.1.100:5000
   - Login: admin / Admin@123
   - Start typing QR codes manually or use camera
